Overleaf package for the MAML-Select IEEE TAI letter

Upload this entire folder as one Overleaf project.

Recommended Overleaf settings:
1. Compiler: pdfLaTeX
2. Main document for the paper: Latex_File_Letter.tex
3. Bibliography file: References_letter.bib

To compile the response letter:
Set the Overleaf main document to Response_to_Reviewers.tex.

To compile the supplementary material:
Set the Overleaf main document to Supplementary_Material.tex.
